# Distinctive Fonts

Place distinctive .woff2 font files here to avoid generic AI aesthetics.

## Recommended Fonts to Add

- Authentic Sans
- Departure Mono
- GT Flexa
- Switzer
- Sohne
- Any distinctive foundry fonts

## Format

Use .woff2 format for optimal web performance.

## Usage

The skill will reference these fonts when creating UI. Users can also specify favorite fonts in `.claude/ui-designer.local.md`.
