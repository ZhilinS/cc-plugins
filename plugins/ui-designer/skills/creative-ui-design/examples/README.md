# UI Design Examples

Place screenshot images (.png, .jpg) of exceptional UI design here.

## What to Include

Screenshots demonstrating:
- Bold, creative color usage
- Distinctive typography hierarchies
- Atmospheric backgrounds
- Creative layouts
- Thoughtful animations
- Context-specific character

## Purpose

Claude analyzes these screenshots to extract aesthetic principles and apply them contextually to UI work. These serve as visual references for what "good design" looks like.

## Usage

The skill will read and analyze these screenshots when creating UI. Users can also provide their own screenshot references in `.claude/ui-designer.local.md`.
